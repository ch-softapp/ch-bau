# ch-Bau v3.5.0

Komplette Progressive Web App für Bauprofis - 11 Gewerke offline verfügbar.

## 📋 Gewerke

- ⚡ **Elektro** - Berechnungen, Symbole, Normen, Blitzschutz
- 🔧 **HLSK** - Rohre, MAG, Hydraulik, Heizlast
- 🏗️ **Statik** - Träger, Stützen, Bewehrung, Verbindungen
- 🔥 **Brandschutz** - Feuerwiderstand, Flucht, Kabel, Löschwasser
- ⚠️ **Arbeitsschutz** - PSA, Gefährdung, Unterweisungen, Grenzwerte
- 📋 **Baurecht** - VOB/B, Nachträge, HOAI, LBO
- 🌱 **GEG/Energie** - U-Wert, Dämmung, Lüftung, Hydraulischer Abgleich
- ☀️ **PV/Solar** - Anlage, Module, Speicher, Ertrag
- 🔊 **Schallschutz** - DIN 4109, Luft-/Trittschall
- 🏠 **KNX/Smart Home** - Topologie, ETS, DALI, Diagnose
- 🚧 **Tiefbau** - Erdarbeiten, Gräben, Kanal, Entwässerung

## 🚀 Installation

1. ZIP entpacken auf Webserver oder lokal
2. Im Browser öffnen
3. "Zum Startbildschirm hinzufügen" für PWA-Installation
4. Funktioniert vollständig offline!

Diese App dient ausschließlich als **Entscheidungshilfe**. Alle Angaben müssen eigenverantwortlich geprüft und mit aktuellen Normen abgeglichen werden.
